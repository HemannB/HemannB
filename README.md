## AirSniffer — a project I'm proud of

A full air quality monitoring system built from scratch:
- Hardware: sensors, wiring, ESP32 microcontroller
- Firmware: written and debugged via serial
- API: JavaScript, collecting and serving JSON data from the firmware
- Dashboard: built by me, data persisted in a remote PostgreSQL database

Three layers. One person. Good learning.
